AppArmor on Ubuntu

Due to a sever case of mental retardation in the Canonical teams, Ubuntu enables AppArmor protection on CUPS despite knowing very well that it will prevent some (many) printers from working correctly.

Disabling AppArmor on CUPS

  sudo apt-get install apparmor-utils
  
  sudo aa-complain cupsd
