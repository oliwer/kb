Firefox Tips

Privacy

Mozilla is not a non-profit organization anymore. The good old days are over.

Google Safe Browsing

Malware detection is useless on Linux anyway.

  browser.safebrowsing.enabled
  browser.safebrowsing.downloads.enabled
  browser.safebrowsing.malware.enabled

Statistics and health reports

Collected by Mozilla. Like if bug reports were not enough.

  datareporting.healthreport.service.enabled
  datareporting.healthreport.uploadEnabled
  toolkit.telemetry.enabled

WebRTC

Leaks your private IP (source).

  media.peerconnection.enabled

DRM

Firefox uses a binary blob to play medias encrypted with DRM.

  media.eme.enabled
  media.gmp-eme-adobe.enabled

Hello

Connects to Telefonica.

  loop.enabled

Pocket

We have bookmarks for that.

  browser.pocket.enabled

Geolocation

Your IP is enough.

  geo.enabled

Tracking Protection

This will block sites which are known to track users. As a side effect, it makes pages load faster!

  privacy.trackingprotection.enabled

1024-bit Diffie-Hellman

Research suggests that the NSA has developed the capability to decrypt a large number of HTTPS, SSH, and VPN connections using an attack on common implementations of the Diffie-Hellman key exchange algorithm with 1024-bit primes.

The following options should be set to "false".

  security.ssl3.dhe_rsa_aes_128_sha
  security.ssl3.dhe_rsa_aes_256_sha

The EFF has set up a website to test your browser TLS configuration: How's My SSL

Performance

Choppy scrolling

Enabling this option increases animation speed which reduces scrolling lags.

  layout.frame_rate.precise

Also, if your mouse scrolls too slowly, raise this option to a value like 40.

  mousewheel.min_line_scroll_amount

Session history

By default, each opened tab will keep its own history of the last 50 visited pages. This is too much. Set it to 10 or 15.

  browser.sessionhistory.max_entries
